MindGuide-AI is an AI-powered mental wellness assistant that predicts a user's **emotional state** and **stress level** based on behavioral, contextual, and textual inputs.

It combines Natural Language Processing (NLP) with structured data to provide personalized insights and recommendations.

---

## 🚀 Features

- Emotion Prediction (calm, neutral, overwhelmed, etc.)
- Stress Level Detection (low, medium, high)
- Journal Text Analysis using NLP
- Personalized Suggestions based on predictions
- Interactive UI built with Streamlit

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy
- Joblib
- NLP Vectorization (TF-IDF / CountVectorizer)
